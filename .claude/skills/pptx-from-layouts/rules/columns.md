# Columns and Cards

**Core Principle: Columns create structure.** Use `[Column N: Header]` for process/comparison, `[Card N: Title]` for discrete items.

## Column Syntax

```markdown
[Column 1: Discover]
- Stakeholder interviews
- Competitive audit
**Output:** Insight synthesis
[Image: research in progress]

[Column 2: Define]
- Workshop facilitation
- Persona development
**Output:** Strategic framework
[Image: workshop activity]
```

## Card Syntax

```markdown
[Card 1: Strategic Framework]
Prioritized opportunities mapped to objectives

[Card 2: Consumer Segmentation]
Actionable personas with targeting recommendations
```

## Image Placeholders

```markdown
[Column 1: Phase Name]
- Activity 1
- Activity 2
[Image: description of needed image]
```

Images can appear inside columns or at slide level.

## Column Guidelines

- Keep headers short: 2-3 words max
- Limit to 4 bullets per column
- Maximum 10 items total across all columns
- Use consistent header lengths across columns

## FORBIDDEN

- NEVER use inconsistent header lengths (2-3 words max)
- NEVER exceed 4 bullets per column
- NEVER mix column and card syntax on same slide
- NEVER omit the column number
- NEVER use columns for non-parallel content
- NEVER exceed 10 items total across all columns
